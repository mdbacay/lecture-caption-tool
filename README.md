# Lecture Caption Tool

A real-time lecture captioning app that transcribes audio live during lectures and generates a summary at the end of the session.

## What it does
- Captures audio from your microphone
- Displays live captions on screen as you speak
- Generates a summary of the lecture when you stop recording
- Lets you download the full transcript

## Tech Stack
- **Frontend:** React
- **Backend:** Node.js + Express
- **Database:** MongoDB
- **Speech-to-Text:** Google Speech-to-Text API
- **Summaries:** Anthropic (Claude) API
